# Specification: Weboldal alapstruktúra és MVP felépítése

## Cél
A projekt alapstruktúrájának létrehozása és az MVP (Minimum Viable Product) funkcióinak (kezdőlap, rólunk, szolgáltatások, kapcsolat) kifejlesztése.

## Követelmények
- Next.js (App Router) és Tailwind CSS alapok inicializálása.
- Alapvető SEO és layout konfiguráció.
- 4 fő szekció/oldal felépítése:
  - Hero szekció (Nyitó).
  - Szolgáltatások (Grid/Kártyák).
  - Rólunk (Pohánka Józsefné bemutatkozás).
  - Kapcsolati űrlap (Server Action backenddel).
- Reszponzív design és gyors betöltődés.
